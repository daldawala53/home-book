🔧 One-time Firebase setup (5 minutes)

1
Go to console.firebase.google.com → Add project (name it "homebooks")
2
Authentication → Sign-in method → Google → Enable → add your domain to Authorized Domains
3
Firestore Database → Create database → Production mode
4
Firestore → Rules tab → replace with: rules_version='2'; service cloud.firestore { match /databases/{database}/documents { match /users/{uid}/{document=**} { allow read, write: if request.auth.uid == uid; } } } → Publish
5
Project Settings → Your apps → Add web app → copy the firebaseConfig values
6
Open this HTML file in a text editor → find const FIREBASE_CONFIG at the top of the script → paste your values → save
7
Host this file anywhere (GitHub Pages, Netlify, or even a local server) and open on any device — sign in with the same Google account to sync
